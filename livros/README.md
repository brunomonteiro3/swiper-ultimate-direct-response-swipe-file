> Além dos clássicos dos [grandes autores](../autores/README.md), existe uma segunda camada de leitura obrigatória: livros que aprofundam princípios específicos, atacam ângulos que os clássicos não cobrem, ou apresentam frameworks práticos prontos para uso.

## Como usar esta seção

Cada arquivo cobre:
- O que o livro ensina e por que é relevante
- Os conceitos mais aplicáveis ao copy moderno
- Como conectar com o que você já viu nos [grandes autores](../autores/README.md)
- Onde aprofundar no Swiper quando há material correspondente

**Sequência sugerida de leitura:**

```
1. Breakthrough Advertising (Schwartz) — fundação de tudo
2. Brain Audit (D'Souza) — entender os bloqueios do comprador
3. Cash Copy (Lant) — orientação prática de benefício
4. Wizard of Ads (Williams) — persuasão não convencional
5. Power Copywriting (Lewis) — técnica de mala direta e copy direto
6. Crypto Copywriting Secrets (Settle) — e-mail e tom de voz moderno
```
