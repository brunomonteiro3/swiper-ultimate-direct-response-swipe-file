## O que é isso?

Este repositório é um **guia de estudos de direct response e copywriting** organizado por formato, componente e princípio.

Cada seção cobre um elemento do funil de vendas. Cada artigo conecta a teoria a **exemplos reais analisados no [Swiper](https://swiper.com.br)** — o maior acervo de copywriting em português, com centenas de anúncios, e-mails, landing pages e VSLs destrinchados.

**Para quem é:**
- Copywriters que querem referência rápida por formato
- Gestores de tráfego que precisam entender o copy que gerenciam
- Donos de negócio que escrevem o próprio copy
- Estudantes de marketing de resposta direta

## 🎬 VSL — Video Sales Letter

> Uma VSL é um vídeo cuja única função é vender. Ela controla o ritmo de entrega da informação, elimina a possibilidade de o avatar "rolar para o preço" e combina argumento + emoção de uma forma que texto raramente alcança.

📄 [Visão geral da VSL →](./vsl/README.md)

| Componente | O que cobre | Arquivo |
|------------|-------------|---------|
| **Hook / Abertura** | Primeiros 5–10s. Loop de vídeo. CTA contextualizado no play. | [hook.md](./vsl/hook.md) |
| **Lead** | A miniatura da VSL. Estrutura da lead de $10M trecho a trecho. | [lead.md](./vsl/lead.md) |
| **Mecanismo Único** | "Por que só isso funciona." Nome proprietário, invalidar soluções anteriores. | [mecanismo-unico.md](./vsl/mecanismo-unico.md) |
| **Retenção** | Play rate, pacing, loops de antecipação, player dedicado. | [retencao.md](./vsl/retencao.md) |
| **Oferta** | Stack de valor, upsell, momento pós-compra, tipos de upsell. | [oferta.md](./vsl/oferta.md) |
| **CTA** | Dois botões, urgência real, garantia antes do clique. | [cta.md](./vsl/cta.md) |
