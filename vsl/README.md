# VSL — Video Sales Letter

Uma VSL é um vídeo cuja única função é vender. Diferente de um vídeo de conteúdo, ela tem estrutura de direct response: prende, convence e converte.

## Anatomia de uma VSL

```
[Hook]  →  [Lead]  →  [Problema/Agitação]  →  [Mecanismo Único]
     →  [Prova]  →  [Oferta]  →  [CTA]
```

A **lead** é a parte mais crítica: ela funciona como uma VSL condensada. Se a pessoa não passar da lead, nada mais importa.

## Por que VSL funciona

- Vídeo reduz atrito cognitivo (mais fácil consumir do que ler)
- O copywriter controla o ritmo de entrega da informação
- Elementos visuais e áudio ampliam o impacto emocional
- Impossibilidade de "rolar para o preço" (quando bem feita)

## Erros mais comuns

- **Lead longa demais** antes de qualquer promessa ou mistério
- **Player errado** (YouTube tem pontos de fuga; use player dedicado)
- **Depoimentos genéricos** sem contexto visual ou interação
- **CTA genérico** ("Clique aqui") em vez de contextualizado para o público
